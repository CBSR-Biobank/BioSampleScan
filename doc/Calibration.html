Calibration Process

 

Before you begin scanning test tubes you need to select the proper driver and calibrate the appropriate plates.

 

To select your scanner driver go scanner->select source.

The driver you select MUST be either of type WIA or TWAIN.

Consult you scanners manual to determine whether you have a TWAIN driver for your scanner.

WIA is easily identifiable as it is prefixed in the ‘select source’ dialog box.

 

The next step is the calibration process, to get to this dialog box go

scanner->scanner configuration.

 

The  ‘Set Plate Mode’ section is used to select the number of active plates you will be working with.

(eg. If you are using a 3-plate setup, select ratio box labeled 3.)

 

The  ‘Driver Type’ section is used to configure the calibration process towards your appropriate type of driver. You MUST select the proper driver type you have selected in the ‘select source’ dialog box.

(NOTE: This setting is entirely dependant on the driver you selected in the ‘select source’  dialog box)

 

The ‘DPI’ setting ranges from 1 to 600.

(NOTE: It is recommended that for speed and quality you select a dpi of ~300)

 

The ‘Brightness’ and ‘Contrast’ settings range from -1000 to 1000.

(NOTE: Brightness and contrast only work on TWAIN drivers)

 

For each plate that you wish to setup, you must enter the appropriate offsets in the text boxes provided below.

 

For the quick calibration method, click the ‘Edit’ button. A confirmation dialog will show up, by clicking OK: this application will rescan a new image for the graphical calibration process. To select outline your plate, first click in the top-left corner and then in the bottom-right corner.

eg. When I place a plate in the top-right corner (using a WIA driver) I use the following values [ Plate 1:  Top: 0.2198 Left: 0.44 Height: 2.9175 Width: 4.22 ]

(NOTE!!!!: You MUST use a plate full with 96 test tubes for calibration)

(NOTE: It is recommended that you provide a buffer space of around 1/16th of an inch around your plate.)

(NOTE: The plate offsets are in inches)

 

Once you are satisfied with your settings, click ‘Apply’. Any changes to the plate offsets will result in a calibration process to be applied to them. Upon successful calibration an information dialog will popup reporting it has found all 96 test tubes.
